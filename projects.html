My most recent projects

Hometown Solo Project is an impressive web application created using HTML and CSS. This project marks a significant milestone for me as it represents my first website app developed in Scrimba. From building the initial repository to successfully deploying it, this project has been a fantastic learning experience. Throughout the process, I had the opportunity to explore and utilize various tools that were new to me. Notably, I discovered the power and creativity of tools like Coolors for color palettes, Netlify for deployment, and GitHub for version control. The project has been an exciting journey, and I thoroughly enjoyed the process of bringing it to life.

Conversion App Project is a remarkable web application that goes beyond HTML and CSS by incorporating the power of JavaScript. Designed to simplify unit conversions, this app stands out as my first project involving multiple programming languages. In addition to HTML and CSS, I embraced the versatility of JavaScript to enhance the functionality and interactivity of the application. Moreover, I expanded my skill set by utilizing Figma, a powerful design tool that allowed me to gather precise requirements for this project. With this project, I gained valuable experience in leveraging different technologies to create practical and user-friendly web applications.

Foodagram is Inspired by the popular Instagram social media platform, the Foodagram web app is an exciting project that adds a creative twist to the concept. While the goal was to clone Instagram, Scrimba introduced a unique spin by creating Oldagram. However, in pursuit of personal creativity, I decided to customize the platform to focus specifically on food-related content, thus transforming it into Foodagram. By utilizing HTML, CSS, and JavaScript, I brought this food-centric social media experience to life. Recognizing the widespread popularity of sharing food-related posts on social media platforms, I found immense joy in developing and designing Foodagram. It allowed me to explore my passion for food while applying my technical skills in a creative manner.
